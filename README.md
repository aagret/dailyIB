# Daily IB Data Tool

## Project Goal
Automated R script for daily processing and analysis of Interactive Brokers (IB) data.

## Usage
Daily data extraction and processing tool for Interactive Brokers account data.

### Running the Script
```r
# Open in RStudio and run the main script
source("main.R")
```

### Features
- Daily data extraction from IB
- Automated data processing
- Report generation

### Requirements
- R with required packages
- Interactive Brokers account and API access
- Proper authentication credentials
